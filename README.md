trivago/yay
===

Gamification done simple.

## Preparation

### Build
```
$ make build
```
### Installation
```
$ make install
```

## Usage

### Help
```
$ make
```

### Run the application
```
$ make run
$ open http://localhost:50080/api/doc
```

### Import demo data
```
$ make import-demo
```

### Run the tests
```
$ make test
```
